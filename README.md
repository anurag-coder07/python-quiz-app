# Python Quiz App

A simple command-line quiz app built with Python. It asks multiple-choice questions, checks the answers, and shows the final score with a category breakdown.

## Features

- multiple-choice Python questions
- instant feedback after each answer
- final score summary
- category-wise results
- best score saved locally

## Run

```bash
python main.py
```

## Tests

```bash
python -m unittest discover -s tests
```
