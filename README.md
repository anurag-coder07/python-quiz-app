# Python Quiz App

A command-line quiz app built with Python. It asks multiple-choice questions, checks the answers, and shows the final score at the end.

## Features

- multiple-choice Python questions
- instant feedback after each answer
- final score summary
- score by category
- best score saved locally

## Run

```bash
python main.py
```

## Tests

```bash
python -m unittest discover -s tests
```
